# Governance
Use the menu on the left to navigate through the governance frameworks. You can also use the Search function (Ctrl+K) to search for a relevent term.

BC Government uses the CANdy Network for some digital trust functionality and adheres to this [CANdy Governance Framework](https://iccs-isac.github.io/Gouvernance-CICAN-ICDTC-Governance/CANdy%20Network/CANdy-Network-Governance-Framework).
